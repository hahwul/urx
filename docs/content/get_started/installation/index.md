### From Cargo

```bash
cargo install urx
```

### From Homebrew

```bash
brew install urx
```

### From Source

```bash
git clone https://github.com/hahwul/urx.git
cd urx
cargo build --release
```

Binary location: `target/release/urx`

### From Docker

```bash
docker pull ghcr.io/hahwul/urx:latest
```
